INTAL - INTegers of Arbitrary Length

In C/C++, the maximum value for Unsigned Long Int is 18446744073709551615, a 20-digit number. While languages such as C++/Java support BigInteger classes (100 digit numbers). C does not include such support by default. This project seeks to add that support to the C language, as well as basic arithmetic operations such as comparison, addition, subtraction, and multiplication, as well as applications such as Factorial, Fibonacci, and so on.
